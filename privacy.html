Privacy Policy for birdvirus bot

Effective Date: July 2026

This privacy policy outlines how the birdvirus bot Discord application handles data within your server.

  • Data Access & Processing: To provide context-aware AI features, the bot reads the recent message history of the specific channel where the command is executed. This history is processed transiently in active memory to generate accurate responses.
  • Data Collection: The bot does not permanently store, log, or harvest user data, full server message logs, or personal identification details on external databases. Channel history is cleared from runtime memory immediately after the AI interaction completes.
  • Data Deletion: Because no data is stored long-term, there is no persistent user data to manage or delete. Removing the bot from your server immediately terminates all active process contexts.
